What is the DOM?
DOM = Document Object Model is an interface for HTML documents and everything in the document is represented as nodes. Programmers can manipulate the DOM to change style and content.


What is an event?
Events are what allow JS to put event handlers on elements in the HTML


What is an event listener?
This represents an object that can handle events from a particular event target, e.g., EventListener.handleEvent()


What is a Constructor?
This is a method for creating an object within a class.
